csv2xlsx
========

Converts CSV from stdin into XLSX to stdout. Also optionally converts columns to integers
or datetimes and adjusts column widths.

Installation
------------

.. code-block:: sh

    pip install csv2xlsx

Usage
-----

.. code-block:: sh

    csv2xlsx 'utf-8' test < ~/test.csv > ~/test.xlsx
